# sweat-sentry
Real-time environmental monitoring system designed to tell me exactly how bloody humid and warm it is.

# Hardware
## Microcontrollers
- ESP32

## Sensors (humidity and temperature)
- DHT22

## Displays
- 16x2 LCD with I2C for displaying data

## Storage
- SD Card to log data

# Tools
- clang
- C++20
- CMake
- GNU/Make
